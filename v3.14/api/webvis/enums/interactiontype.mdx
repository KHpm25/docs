The available interaction types. They determine if a target was picked or interacted with by ID.

#### See

* [InteractionData](/v3.14/api/webvis/interfaces/interactiondata)
* [InteractionAPI](/v3.14/api/webvis/interfaces/interactionapi)

## Enumeration Members

### BY\_ID

BY\_ID: 1

Target is interacted with by ID.

### PICKING

PICKING: 0

Target is interacted with via picking.
