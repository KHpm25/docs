Data associated with a member.

Can be requested via MemberAPI.requestMemberProperties.

## Properties

### `Optional`name

name?: string

The name of the member.

Use MemberAPI.setMemberName to change the name.

### `Optional`profile

profile?: MemberProfile

Custom data associated with the member.

#### See

MemberAPI.setMemberProfileEntry and MemberAPI.deleteMemberProfileEntry
for modifying profile entries.

### `Optional`role

role?: [MemberRole](/v3.14/api/webvis/enums/memberrole)

The role of the member in the space.

#### See

[MemberAction.PROMOTE](/v3.14/api/webvis/enums/memberaction#promote) and [MemberAction.DEMOTE](/v3.14/api/webvis/enums/memberaction#demote) for changing the role of a member.

### `Optional`state

state?: [MemberState](/v3.14/api/webvis/enums/memberstate)

The state of the member.
