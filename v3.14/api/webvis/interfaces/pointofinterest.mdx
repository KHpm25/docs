Defines an individual Point of Interest (POI).

## Properties

### `Optional`id

id?: number

The unique identifier for the POI.
Automatically assigned if not specified.

#### Default

```ts
undefined
```

### `Optional`label

label?: string

Text to display next to the POI.

#### Default

```ts
undefined
```

### position

position: [number, number, number]

The 3D position of a POI.
