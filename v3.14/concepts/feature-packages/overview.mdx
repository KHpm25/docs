Use the Feature Packages to navigate the instant3Dhub API landscape and identify the capabilities most relevant to your solution. All APIs are modular and can be freely combined. Select one of the packages below to explore its features in detail, or browse the Feature Map for a quick overview of all features across packages.

  <Card title="Basic" href="/v3.14/concepts/feature-packages/basic">
    Build simple 3D viewing applications and interact with 3DSpaces.
  </Card>
  <Card title="Advanced" href="/v3.14/concepts/feature-packages/advanced">
    Support complex engineering workflows and advanced scenarios.
  </Card>
  <Card title="Expert" href="/v3.14/concepts/feature-packages/expert">
    Enable immersive AR and VR use cases.
  </Card>
</CardGroup>

## Feature Map

This overview maps every feature grouped by functional categories to the Basic, Advanced, or Expert tier. Click on a feature to learn more, or use the package cards above to explore a tier in detail.

<Note>
Features marked with Experimental are not yet generally available (GA) and may still change.
</Note>
