The properties associated with the magnifier of a viewer.

#### See

ViewerMagnifierAPI

## Properties

### `Optional`enabled

enabled?: boolean

Specifies the enabled state of the magnifier.

#### Default

```ts
true
```

### `Optional`position

position?: [number, number]

Specifies the center x- and y-position of the magnifier in pixels.

#### Default

```ts
[0,0]
```

### `Optional`roundness

roundness?: number

Specifies the roundness of the magnifier between 0 and 100.
0 means a square magnifier, 100 means a circle.

#### Default

```ts
100
```

### `Optional`size

size?: [number, number]

Specifies size of the magnifier in pixels.

#### Default

```ts
[50,50]
```

### `Optional`zoomLevel

zoomLevel?: number

Specifies zoom level of the magnifier.

#### Default

```ts
3
```
