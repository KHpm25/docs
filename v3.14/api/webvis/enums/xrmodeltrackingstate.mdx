The state of the model tracking.

## Enumeration Members

### CRITICAL

CRITICAL: 1

The model is snapped but was not found in the frame

### LOST

LOST: 2

The model is not snapped and not found

### TRACKING

TRACKING: 0

The model is snapped and was found in the frame
