The capabilities of the XR system. The capabilities are used to determine which functionalities are supported by the XR system.

## Enumeration Members

### SUPPORTS\_DEVICE\_IMAGE\_SOURCE

SUPPORTS\_DEVICE\_IMAGE\_SOURCE: "supportsDeviceImageSource"

The XR system supports streaming XR images from the local device.

#### See

* [XRConfiguration.imageSourceConfig](/api/webvis/interfaces/xrconfiguration#imagesourceconfig)
* [XRImageSourceConfigDevice](/api/webvis/interfaces/xrimagesourceconfigdevice)

### SUPPORTS\_DEVICE\_SCREENSHOTS

SUPPORTS\_DEVICE\_SCREENSHOTS: "supportsDeviceScreenshots"

The XR system is capable of providing device screenshots.

#### See

* [XRConfiguration.deviceScreenshotsEnabled](/api/webvis/interfaces/xrconfiguration#devicescreenshotsenabled)
* [RealityAPI.requestXRDeviceScreenshot](/api/webvis/interfaces/realityapi#requestxrdevicescreenshot)

### SUPPORTS\_MODEL\_TRACKING

SUPPORTS\_MODEL\_TRACKING: "supportsModelTracking"

The XR system supports modeltracking.

#### See

* [XRConfiguration.modelTrackingEnabled](/api/webvis/interfaces/xrconfiguration#modeltrackingenabled)
* [RealityAPI.enterXRInitMode](/api/webvis/interfaces/realityapi#enterxrinitmode)

### SUPPORTS\_PLAYBACK

SUPPORTS\_PLAYBACK: "supportsPlayback"

The XR system supports playback of recorded XR images.

#### See

* [XRConfiguration.imageSourceConfig](/api/webvis/interfaces/xrconfiguration#imagesourceconfig)
* [XRImageSourceConfigPlayback](/api/webvis/interfaces/xrimagesourceconfigplayback)
* [RealityAPI.setXRPlaybackSource](/api/webvis/interfaces/realityapi#setxrplaybacksource)
* [RealityAPI.startXRPlayback](/api/webvis/interfaces/realityapi#startxrplayback)

### SUPPORTS\_RAY\_CASTING

SUPPORTS\_RAY\_CASTING: "supportsRayCasting"

The XR system is capable of generating ray casts.

#### See

* [XRConfiguration.rayCastingEnabled](/api/webvis/interfaces/xrconfiguration#raycastingenabled)
* [RealityAPI.anchorXR](/api/webvis/interfaces/realityapi#anchorxr)
