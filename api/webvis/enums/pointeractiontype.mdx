#### See

[PointerInfo](/api/webvis/interfaces/pointerinfo)

## Enumeration Members

### ANY

ANY: 8

Placeholder for any pointer action type.

### CLICKED

CLICKED: 1

Represents a click action.

### DOUBLECLICKED

DOUBLECLICKED: 4

Represents a double click action.

### NONE

NONE: 0

No pointer action.

### PRESSED

PRESSED: 2

Represents a press action.

### RELEASED

RELEASED: 3

Represents a release action.

### TOUCH\_DOUBLE\_PICKED

TOUCH\_DOUBLE\_PICKED: 7

Represents a touch double pick action.

### TOUCH\_PICKED

TOUCH\_PICKED: 5

Represents a touch pick action.

### TOUCH\_PRESSED

TOUCH\_PRESSED: 6

Represents a touch press action.
