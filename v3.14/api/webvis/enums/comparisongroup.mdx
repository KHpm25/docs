Describes, if a node is part of a ColorComparison and if yes, in which group.

## Enumeration Members

### A

A: 1

Group A for color comparison (see [ViewerAPI.enterColorCompareMode](/v3.14/api/webvis/interfaces/viewerapi#entercolorcomparemode)).

### B

B: 2

Group B for color comparison (see [ViewerAPI.enterColorCompareMode](/v3.14/api/webvis/interfaces/viewerapi#entercolorcomparemode)).

### NONE

NONE: 0

### XR\_EDGE

XR\_EDGE: 3

Group for XR edge comparison (see [ViewerXREdgeCompareAPI](/v3.14/api/webvis/interfaces/viewerxredgecompareapi)).
