A handle for a topological entity, which can be used with the
TopologyAPI. The handle represents either a shape or an
entity within a shape. To get a more detailed
[descriptor](/v3.14/api/webvis/types/topologydescriptor) of the topological entity,
the function TopologyAPI.requestTopologyDescriptor can
be used. Besides performing actions such as selecting the
topological entity, the handle can also be used together with
the MeasurementAPI to perform different kinds of
measurements.

#### See

* [TopologyDescriptor](/v3.14/api/webvis/types/topologydescriptor)
* TopologyAPI.requestTopologyDescriptor
* TopologyAPI
* MeasurementAPI
