The result of enabling or disabling a layer filter.

#### See

* LayerFilterAPI
* LayerFilterAPI.setLayerFilterEnabled

## Properties

### hasChanged

hasChanged: boolean

Indicates if the layer filter has changed its state.

### layerFilterName

layerFilterName: string

The name of the layer filter.

### value

value: boolean

The new enabled state of the layer filter.
