A list of predefined view directions that are understood by the viewer
for instance in [ViewerAPI.fitViewToDirection](/api/webvis/interfaces/viewerapi#fitviewtodirection) or
[ViewerAPI.fitViewToVolumeFromDirection](/api/webvis/interfaces/viewerapi#fitviewtovolumefromdirection). Note, that the viewer's
interpretation of these directions depends on the current ViewPerspective.
