This enum lists all event types that are available in the webvis [EventAPI](/v3.14/api/webvis/interfaces/eventapi). Each event
type corresponds to a specific subclass of the [WebVisEvent](/v3.14/api/webvis/classes/webvisevent) base class, as linked in the
description of each enum entry. An event's type can be queried using its
[type](/v3.14/api/webvis/classes/webvisevent#type) property.

#### See

[EventAPI](/v3.14/api/webvis/interfaces/eventapi)
